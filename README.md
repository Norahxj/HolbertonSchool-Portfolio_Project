# Asalah - Portfolio Project

## Project Overview

**Asalah** is a gamified value-based financial literacy platform designed for Saudi families. The project aims to help parents teach children responsible financial habits through meaningful tasks, rewards, and a points-based system called **Noor Points**.

The platform focuses on combining financial literacy, family values, cultural relevance, and gamification in one child-friendly and parent-controlled experience.

## MVP Concept

The MVP will include two main interfaces:

- **Parent Interface:** Allows parents to create child profiles, assign tasks, approve completed tasks, manage points, and create rewards.
- **Child Interface:** Allows children to view tasks, complete them, earn Noor Points, track progress, and request rewards.

## Target Users

- Saudi parents aged 25–45
- Children aged 6–15

## Main Features

- User registration and login
- Child profile creation
- Task creation and assignment
- Task approval and rejection
- Noor Points system
- Reward creation and redemption
- Basic progress tracking
- Parent and child dashboards
